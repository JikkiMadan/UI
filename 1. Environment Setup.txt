Download & Install Visual Studio Code
-------------------------------------

- Google for Visual Studio Code
https://code.visualstudio.com/
> Download for Windows - click

VSCodeUserSetup-x64-1.42.0.exe

Install It

Default Path of Installation : C:\Users\LENOVO\AppData\Local\Programs\Microsoft VS Code

Right click on Code Icon in Desktop - 
Shortcut : "C:\Users\<<ur user name>>\AppData\Local\Programs\Microsoft VS Code\Code.exe"  --disable-gpu

Workspace: is the directory where the programs are being saved

> To Change Editor Settings
  File (menu) > Preferences > Settings

> Integrate "Open in Browser" Extension
   View (menu) > Extensions 
  
   Search for "Open in Browser" > Install

> Integrate Live Server 
  - Launch a development local Server with live reload feature for static & dynamic pages

  Search for Live Server > Install

> Integrate "Auto Save" Extension
 - With this extension you DON'T NEED TO press Ctrl + S anymore
 Search for "Auto Save" Extension
